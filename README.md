## 나의 자산 정보 서비스

제가 가진 + / - 자산 항목을 요약해서 보여주고, 생성, 수정, 삭제할 수 있는 서비스 입니다.

### 프로젝트 실행

```
  yarn

  yarn start:server // json server 구동
  yarn dev // SPA 실행
  yarn storybook // 스토리북 확인
  yarn test // 테스트

```

<br />
